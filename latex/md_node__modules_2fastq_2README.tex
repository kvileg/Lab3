Creates a new queue with {\ttfamily Promise} apis. It also offers all the methods and properties of the object returned by \doxylink{md_node__modules_2fastq_2README_fastqueue}{\`{}fastqueue\`{}} with the modified \doxylink{md_node__modules_2fastq_2README_pushPromise}{\`{}push\`{}} and \doxylink{md_node__modules_2fastq_2README_unshiftPromise}{\`{}unshift\`{}} methods.

Node v10+ is required to use the promisified version.

Arguments\+:
\begin{DoxyItemize}
\item {\ttfamily that}, optional context of the {\ttfamily worker} function.
\item {\ttfamily worker}, worker function, it would be called with {\ttfamily that} as {\ttfamily this}, if that is specified. It MUST return a {\ttfamily Promise}.
\item {\ttfamily concurrency}, number of concurrent tasks that could be executed in parallel.
\end{DoxyItemize}
