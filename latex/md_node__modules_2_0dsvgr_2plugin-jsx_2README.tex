{\ttfamily @svgr/plugin-\/jsx} consists in three phases\+:


\begin{DoxyItemize}
\item Parsing the SVG code using \href{https://github.com/Rich-Harris/svg-parser}{\texttt{ svg-\/parser}}
\item Converting the \href{https://github.com/syntax-tree/hast}{\texttt{ HAST}} into a \href{https://github.com/babel/babel/blob/master/packages/babel-parser/ast/spec.md}{\texttt{ Babel AST}}
\item Applying \doxysectlink{md_node__modules_2_0dsvgr_2babel-preset_2README}{\`{}@svgr/babel-\/preset\`{}}{0} transformations
\end{DoxyItemize}\hypertarget{md_node__modules_2_0dsvgr_2plugin-jsx_2README_autotoc_md2795}{}\doxysection{\texorpdfstring{Applying custom transformations}{Applying custom transformations}}\label{md_node__modules_2_0dsvgr_2plugin-jsx_2README_autotoc_md2795}
