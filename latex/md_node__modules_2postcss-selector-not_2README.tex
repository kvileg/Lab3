\href{https://github.com/csstools/postcss-plugins/tree/main/plugins/postcss-selector-not}{\texttt{ Post\+CSS Selector Not}} transforms \+:not() W3C CSS level 4 pseudo classes to \+:not() CSS level 3 selectors following the \href{https://www.w3.org/TR/selectors-4/\#negation-pseudo}{\texttt{ Selectors 4 Specification}}.


\begin{DoxyCode}{0}
\DoxyCodeLine{p:not(:first-\/child,\ .special)\ \{}
\DoxyCodeLine{\ \ \ \ color:\ red;}
\DoxyCodeLine{\}}
\DoxyCodeLine{}
\DoxyCodeLine{/*\ becomes\ */}
\DoxyCodeLine{}
\DoxyCodeLine{p:not(:first-\/child):not(.special)\ \{}
\DoxyCodeLine{\ \ \ \ color:\ red;}
\DoxyCodeLine{\}}

\end{DoxyCode}


⚠️ Only lists of simple selectors ({\ttfamily \+:not(.a, .b)}) will work as expected.
