\chapter{Disallow commented out tests (\texorpdfstring{$<$}{<}tt\texorpdfstring{$>$}{>}no-\/commented-\/out-\/tests\texorpdfstring{$<$}{<}/tt\texorpdfstring{$>$}{>})}

\Hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-commented-out-tests_autotoc_md11373}%
 This rule raises a warning about commented out tests. It\textquotesingle{}s similar to no-\/disabled-\/tests rule.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-commented-out-tests_autotoc_md11374}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-commented-out-tests_autotoc_md11374}
The rule uses fuzzy matching to do its best to determine what constitutes a commented out test, checking for a presence of {\ttfamily it(}, {\ttfamily describe(}, {\ttfamily it.\+skip(}, etc. in code comments.

The following patterns are considered warnings\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ describe('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ it('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ test('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{}
\DoxyCodeLine{//\ describe.skip('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ it.skip('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ test.skip('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{}
\DoxyCodeLine{//\ describe['skip']('bar',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ it['skip']('bar',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ test['skip']('bar',\ ()\ =>\ \{\});}
\DoxyCodeLine{}
\DoxyCodeLine{//\ xdescribe('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ xit('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{//\ xtest('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{}
\DoxyCodeLine{/*}
\DoxyCodeLine{describe('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{*/}

\end{DoxyCode}


These patterns would not be considered warnings\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{describe('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{it('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{test('foo',\ ()\ =>\ \{\});}
\DoxyCodeLine{}
\DoxyCodeLine{describe.only('bar',\ ()\ =>\ \{\});}
\DoxyCodeLine{it.only('bar',\ ()\ =>\ \{\});}
\DoxyCodeLine{test.only('bar',\ ()\ =>\ \{\});}
\DoxyCodeLine{}
\DoxyCodeLine{//\ foo('bar',\ ()\ =>\ \{\});}

\end{DoxyCode}
\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-commented-out-tests_autotoc_md11375}{}\doxysubsection{\texorpdfstring{Limitations}{Limitations}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-commented-out-tests_autotoc_md11375}
The plugin looks at the literal function names within test code, so will not catch more complex examples of commented out tests, such as\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ const\ testSkip\ =\ test.skip;}
\DoxyCodeLine{//\ testSkip('skipped\ test',\ ()\ =>\ \{\});}
\DoxyCodeLine{}
\DoxyCodeLine{//\ const\ myTest\ =\ test;}
\DoxyCodeLine{//\ myTest('does\ not\ have\ function\ body');}

\end{DoxyCode}
