 Prevents {\ttfamily expect} statements outside of a {\ttfamily test} or {\ttfamily it} block. An {\ttfamily expect} within a helper function (but outside of a {\ttfamily test} or {\ttfamily it} block) will not trigger this rule.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-standalone-expect_autotoc_md11427}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-standalone-expect_autotoc_md11427}
This rule aims to eliminate {\ttfamily expect} statements that will not be executed. An {\ttfamily expect} inside of a {\ttfamily describe} block but outside of a {\ttfamily test} or {\ttfamily it} block or outside a {\ttfamily describe} will not execute and therefore will trigger this rule. It is viable, however, to have an {\ttfamily expect} in a helper function that is called from within a {\ttfamily test} or {\ttfamily it} block so {\ttfamily expect} statements in a function will not trigger this rule.

Statements like {\ttfamily expect.\+has\+Assertions()} will NOT trigger this rule since these calls will execute if they are not in a test block.

Examples of {\bfseries{incorrect}} code for this rule\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ in\ describe}
\DoxyCodeLine{describe('a\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ expect(1).toBe(1);}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{//\ below\ other\ tests}
\DoxyCodeLine{describe('a\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ it('an\ it',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ \ \ expect(1).toBe(1);}
\DoxyCodeLine{\ \ \});}
\DoxyCodeLine{}
\DoxyCodeLine{\ \ expect(1).toBe(1);}
\DoxyCodeLine{\});}

\end{DoxyCode}


Examples of {\bfseries{correct}} code for this rule\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ in\ it\ block}
\DoxyCodeLine{describe('a\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ it('an\ it',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ \ \ expect(1).toBe(1);}
\DoxyCodeLine{\ \ \});}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{//\ in\ helper\ function}
\DoxyCodeLine{describe('a\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ const\ helper\ =\ ()\ =>\ \{}
\DoxyCodeLine{\ \ \ \ expect(1).toBe(1);}
\DoxyCodeLine{\ \ \};}
\DoxyCodeLine{}
\DoxyCodeLine{\ \ it('an\ it',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ \ \ helper();}
\DoxyCodeLine{\ \ \});}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{describe('a\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ expect.hasAssertions(1);}
\DoxyCodeLine{\});}

\end{DoxyCode}
