 \begin{quote}
A tiny (304B to 489B) utility to check for deep equality \end{quote}
This module supports comparison of all types, including {\ttfamily Function}, {\ttfamily Reg\+Exp}, {\ttfamily Date}, {\ttfamily Set}, {\ttfamily Map}, {\ttfamily Typed\+Array}s, {\ttfamily Data\+View}, {\ttfamily null}, {\ttfamily undefined}, and {\ttfamily NaN} values. Complex values (eg, Objects, Arrays, Sets, Maps, etc) are traversed recursively.

\begin{quote}
{\bfseries{Important\+:}}
\begin{DoxyItemize}
\item key order {\bfseries{within Objects}} does not matter
\item value order {\bfseries{within Arrays}} {\itshape does} matter
\item values {\bfseries{within Sets and Maps}} use value equality
\item keys {\bfseries{within Maps}} use value equality 
\end{DoxyItemize}\end{quote}
