Verifies that all named imports are part of the set of named exports in the referenced module.

For {\ttfamily export}, verifies that all named exports exist in the referenced module.

Note\+: for packages, the plugin will find exported names from \mbox{[}{\ttfamily jsnext\+:main}\mbox{]} (deprecated) or {\ttfamily module}, if present in {\ttfamily package.\+json}. Redux\textquotesingle{}s npm module includes this key, and thereby is lintable, for example.

A module path that is \href{../../README.md\#importignore}{\texttt{ ignored}} or not \href{https://github.com/bmeck/UnambiguousJavaScriptGrammar}{\texttt{ unambiguously an ES module}} will not be reported when imported. Note that type imports and exports, as used by \href{https://flow.org/}{\texttt{ Flow}}, are always ignored.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2named_autotoc_md10860}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2named_autotoc_md10860}
