 Check whether a source string looks like an ES6 module. This doesn\textquotesingle{}t actually execute the code, and doesn\textquotesingle{}t check other module types. So source strings without any module loaders returns {\ttfamily false}.

This is just what I need from \href{https://github.com/yahoo/js-module-formats}{\texttt{ https\+://github.\+com/yahoo/js-\/module-\/formats}}, which actually executes the sauce string in a subcontext.\hypertarget{md_node__modules_2is-module_2README_autotoc_md15824}{}\doxysection{\texorpdfstring{API}{API}}\label{md_node__modules_2is-module_2README_autotoc_md15824}
