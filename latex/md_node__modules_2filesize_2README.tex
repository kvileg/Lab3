{\ttfamily filesize.\+partial()} takes the second parameter of {\ttfamily filesize()} and returns a new function with the configuration applied upon execution. This can be used to reduce {\ttfamily Object} creation if you call {\ttfamily filesize()} without caching the {\ttfamily descriptor} in lexical scope.
