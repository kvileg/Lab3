 Please refer to its documentation\+:


\begin{DoxyItemize}

\item \href{https://facebook.github.io/create-react-app/}{\texttt{ User Guide}} – How to develop apps bootstrapped with Create React App.
\end{DoxyItemize}\hypertarget{md_node__modules_2eslint-config-react-app_2README_autotoc_md10253}{}\doxysection{\texorpdfstring{Usage in Create React App Projects}{Usage in Create React App Projects}}\label{md_node__modules_2eslint-config-react-app_2README_autotoc_md10253}
The easiest way to use this configuration is with \href{https://github.com/facebook/create-react-app}{\texttt{ Create React App}}, which includes it by default.

{\bfseries{You don’t need to install it separately in Create React App projects.}}\hypertarget{md_node__modules_2eslint-config-react-app_2README_autotoc_md10254}{}\doxysection{\texorpdfstring{Usage Outside of Create React App}{Usage Outside of Create React App}}\label{md_node__modules_2eslint-config-react-app_2README_autotoc_md10254}
If you want to use this ESLint configuration in a project not built with Create React App, you can install it with the following steps.

First, install this package and ESLint.


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install\ -\/-\/save-\/dev\ eslint-\/config-\/react-\/app\ eslint@\string^8.0.0}

\end{DoxyCode}


Then create a file named {\ttfamily .eslintrc.\+json} with following contents in the root folder of your project\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ "{}extends"{}:\ "{}react-\/app"{}}
\DoxyCodeLine{\}}

\end{DoxyCode}


That\textquotesingle{}s it! You can override the settings from {\ttfamily eslint-\/config-\/react-\/app} by editing the {\ttfamily .eslintrc.\+json} file. Learn more about \href{https://eslint.org/docs/user-guide/configuring}{\texttt{ configuring ESLint}} on the ESLint website.\hypertarget{md_node__modules_2eslint-config-react-app_2README_autotoc_md10255}{}\doxysection{\texorpdfstring{Jest rules}{Jest rules}}\label{md_node__modules_2eslint-config-react-app_2README_autotoc_md10255}
This config also ships with optional Jest rules for ESLint (based on \href{https://github.com/jest-community/eslint-plugin-jest}{\texttt{ {\ttfamily eslint-\/plugin-\/jest}}}).

You can enable these rules by adding the Jest config to the {\ttfamily extends} array in your ESLint config.


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ "{}extends"{}:\ ["{}react-\/app"{},\ "{}react-\/app/jest"{}]}
\DoxyCodeLine{\}}

\end{DoxyCode}
