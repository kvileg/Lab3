 Tests in Jest should be void and not return values.

If you are returning Promises then you should update the test to use {\ttfamily async/await}.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-test-return-statement_autotoc_md11434}{}\doxysection{\texorpdfstring{Rule details}{Rule details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-test-return-statement_autotoc_md11434}
This rule triggers a warning if you use a return statement inside a test body.

\DoxyCodeLine{}
\DoxyCodeLine{//\ valid:}

\DoxyCodeLine{}
