Enforces having one or more empty lines after the last top-\/level import statement or require call.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2newline-after-import_autotoc_md10870}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2newline-after-import_autotoc_md10870}
This rule supports the following options\+:


\begin{DoxyItemize}
\item {\ttfamily count} which sets the number of newlines that are enforced after the last top-\/level import statement or require call. This option defaults to {\ttfamily 1}.
\item {\ttfamily exact\+Count} which enforce the exact numbers of newlines that is mentioned in {\ttfamily count}. This option defaults to {\ttfamily false}.
\item {\ttfamily consider\+Comments} which enforces the rule on comments after the last import-\/statement as well when set to true. This option defaults to {\ttfamily false}.
\end{DoxyItemize}
