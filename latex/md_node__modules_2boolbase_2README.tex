\#boolbase This very simple module provides two basic functions, one that always returns true ({\ttfamily true\+Func}) and one that always returns false ({\ttfamily false\+Func}).\hypertarget{md_node__modules_2boolbase_2README_autotoc_md6889}{}\doxysubsubsection{\texorpdfstring{WTF?}{WTF?}}\label{md_node__modules_2boolbase_2README_autotoc_md6889}
By having only a single instance of these functions around, it\textquotesingle{}s possible to do some nice optimizations. Eg. \href{https://github.com/fb55/CSSselect}{\texttt{ {\ttfamily CSSselect}}} uses these functions to determine whether a selector won\textquotesingle{}t match any elements. If that\textquotesingle{}s the case, the DOM doesn\textquotesingle{}t even have to be touched.\hypertarget{md_node__modules_2boolbase_2README_autotoc_md6890}{}\doxysubsubsection{\texorpdfstring{And why is this a separate module?}{And why is this a separate module?}}\label{md_node__modules_2boolbase_2README_autotoc_md6890}
I\textquotesingle{}m trying to modularize {\ttfamily CSSselect} and most modules depend on these functions. IMHO, having a separate module is the easiest solution to this problem.
