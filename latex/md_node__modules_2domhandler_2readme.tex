Add a {\ttfamily start\+Index} property to nodes. When the parser is used in a non-\/streaming fashion, {\ttfamily start\+Index} is an integer indicating the position of the start of the node in the document. The default value is {\ttfamily false}.\hypertarget{md_node__modules_2domhandler_2readme_autotoc_md9651}{}\doxysection{\texorpdfstring{Option\+: $<$tt$>$with\+End\+Indices$<$/tt$>$}{Option: <tt>withEndIndices</tt>}}\label{md_node__modules_2domhandler_2readme_autotoc_md9651}
Add an {\ttfamily end\+Index} property to nodes. When the parser is used in a non-\/streaming fashion, {\ttfamily end\+Index} is an integer indicating the position of the end of the node in the document. The default value is {\ttfamily false}.\hypertarget{md_node__modules_2domhandler_2readme_autotoc_md9652}{}\doxysection{\texorpdfstring{Option\+: $<$tt$>$normalize\+Whitespace$<$/tt$>$ \+\_\+(deprecated)\+\_\+}{Option: <tt>normalizeWhitespace</tt> \_(deprecated)\_}}\label{md_node__modules_2domhandler_2readme_autotoc_md9652}
