\href{https://github.com/postcss/postcss}{\texttt{ Post\+CSS}} plugin to parse CSS and add vendor prefixes to CSS rules using values from \href{https://caniuse.com/}{\texttt{ Can I Use}}. It is recommended by Google and used in Twitter and Alibaba.

Write your CSS rules without vendor prefixes (in fact, forget about them entirely)\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{::placeholder\ \{}
\DoxyCodeLine{\ \ color:\ gray;}
\DoxyCodeLine{\}}
\DoxyCodeLine{}
\DoxyCodeLine{.image\ \{}
\DoxyCodeLine{\ \ background-\/image:\ url(image@1x.png);}
\DoxyCodeLine{\}}
\DoxyCodeLine{@media\ (min-\/resolution:\ 2dppx)\ \{}
\DoxyCodeLine{\ \ .image\ \{}
\DoxyCodeLine{\ \ \ \ background-\/image:\ url(image@2x.png);}
\DoxyCodeLine{\ \ \}}
\DoxyCodeLine{\}}

\end{DoxyCode}


Autoprefixer will use the data based on current browser popularity and property support to apply prefixes for you. You can try the \href{https://autoprefixer.github.io/}{\texttt{ interactive demo}} of Autoprefixer.

\DoxyCodeLine{::-\/moz-\/placeholder\ \{}
\DoxyCodeLine{\ \ color:\ gray;}
\DoxyCodeLine{\}}
\DoxyCodeLine{::placeholder\ \{}
\DoxyCodeLine{\ \ color:\ gray;}
\DoxyCodeLine{\}}
\DoxyCodeLine{}
\DoxyCodeLine{.image\ \{}
\DoxyCodeLine{\ \ background-\/image:\ url(image@1x.png);}
\DoxyCodeLine{\}}
\DoxyCodeLine{@media\ (-\/webkit-\/min-\/device-\/pixel-\/ratio:\ 2),}
\DoxyCodeLine{\ \ \ \ \ \ \ (min-\/resolution:\ 2dppx)\ \{}
\DoxyCodeLine{\ \ .image\ \{}
\DoxyCodeLine{\ \ \ \ background-\/image:\ url(image@2x.png);}
\DoxyCodeLine{\ \ \}}
\DoxyCodeLine{\}}

\end{DoxyCode}
