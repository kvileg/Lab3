This is plugin for \href{http://marijnhaverbeke.nl/acorn/}{\texttt{ Acorn}} -\/ a tiny, fast Java\+Script parser, written completely in Java\+Script.

It was created as an experimental alternative, faster \href{http://facebook.github.io/react/docs/jsx-in-depth.html}{\texttt{ React.\+js JSX}} parser. Later, it replaced the \href{https://github.com/facebookarchive/esprima}{\texttt{ official parser}} and these days is used by many prominent development tools.\hypertarget{md_node__modules_2acorn-jsx_2README_autotoc_md5158}{}\doxysection{\texorpdfstring{Transpiler}{Transpiler}}\label{md_node__modules_2acorn-jsx_2README_autotoc_md5158}
Please note that this tool only parses source code to JSX AST, which is useful for various language tools and services. If you want to transpile your code to regular ES5-\/compliant Java\+Script with source map, check out \href{https://babeljs.io/}{\texttt{ Babel}} and \href{https://buble.surge.sh/}{\texttt{ Buble}} transpilers which use {\ttfamily acorn-\/jsx} under the hood.\hypertarget{md_node__modules_2acorn-jsx_2README_autotoc_md5159}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md_node__modules_2acorn-jsx_2README_autotoc_md5159}
Requiring this module provides you with an Acorn plugin that you can use like this\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ acorn\ =\ require("{}acorn"{});}
\DoxyCodeLine{var\ jsx\ =\ require("{}acorn-\/jsx"{});}
\DoxyCodeLine{acorn.Parser.extend(jsx()).parse("{}my(<jsx/>,\ 'code');"{});}

\end{DoxyCode}


Note that official spec doesn\textquotesingle{}t support mix of XML namespaces and object-\/style access in tag names (\#27) like in {\ttfamily \texorpdfstring{$<$}{<}namespace\+:Object.\+Property /\texorpdfstring{$>$}{>}}, so it was deprecated in {\ttfamily acorn-\/jsx@3.\+0}. If you still want to opt-\/in to support of such constructions, you can pass the following option\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{acorn.Parser.extend(jsx(\{\ allowNamespacedObjects:\ true\ \}))}

\end{DoxyCode}


Also, since most apps use pure React transformer, a new option was introduced that allows to prohibit namespaces completely\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{acorn.Parser.extend(jsx(\{\ allowNamespaces:\ false\ \}))}

\end{DoxyCode}
