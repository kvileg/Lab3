Reports use of a deprecated name, as indicated by a JSDoc block with a {\ttfamily @deprecated} tag or Tom\+Doc {\ttfamily Deprecated\+:} comment.

using a JSDoc {\ttfamily @deprecated} tag\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ @file:\ ./answer.js}
\DoxyCodeLine{}
\DoxyCodeLine{/**}
\DoxyCodeLine{\ *\ this\ is\ what\ you\ get\ when\ you\ trust\ a\ mouse\ talk\ show}
\DoxyCodeLine{\ *\ @deprecated\ need\ to\ restart\ the\ experiment}
\DoxyCodeLine{\ *\ @returns\ \{Number\}\ nonsense}
\DoxyCodeLine{\ */}
\DoxyCodeLine{export\ function\ multiply(six,\ nine)\ \{}
\DoxyCodeLine{\ \ return\ 42}
\DoxyCodeLine{\}}

\end{DoxyCode}


will report as such\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{import\ \{\ multiply\ \}\ from\ './answer'\ //\ Deprecated:\ need\ to\ restart\ the\ experiment}
\DoxyCodeLine{}
\DoxyCodeLine{function\ whatever(y,\ z)\ \{}
\DoxyCodeLine{\ \ return\ multiply(y,\ z)\ //\ Deprecated:\ need\ to\ restart\ the\ experiment}
\DoxyCodeLine{\}}

\end{DoxyCode}


or using the Tom\+Doc equivalent\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ Deprecated:\ This\ is\ what\ you\ get\ when\ you\ trust\ a\ mouse\ talk\ show,\ need\ to}
\DoxyCodeLine{//\ restart\ the\ experiment.}
\DoxyCodeLine{//}
\DoxyCodeLine{//\ Returns\ a\ Number\ nonsense}
\DoxyCodeLine{export\ function\ multiply(six,\ nine)\ \{}
\DoxyCodeLine{\ \ return\ 42}
\DoxyCodeLine{\}}

\end{DoxyCode}


Only JSDoc is enabled by default. Other documentation styles can be enabled with the {\ttfamily import/docstyle} setting.


\begin{DoxyCode}{0}
\DoxyCodeLine{\#\ .eslintrc.yml}
\DoxyCodeLine{settings:}
\DoxyCodeLine{\ \ import/docstyle:\ ['jsdoc',\ 'tomdoc']}

\end{DoxyCode}
