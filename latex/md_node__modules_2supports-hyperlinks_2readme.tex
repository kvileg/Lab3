 \begin{quote}
Detect whether a terminal emulator supports hyperlinks \end{quote}
Terminal emulators are \href{https://gist.github.com/egmontkob/eb114294efbcd5adb1944c9f3cb5feda}{\texttt{ starting to support hyperlinks}}. While many terminals have long detected URL\textquotesingle{}s and linkified them, allowing you to Command-\/\+Click or Control-\/\+Click them to open a browser, you were forced to print the long unsightly URL\textquotesingle{}s on the screen. As of spring 2017 \href{https://gist.github.com/egmontkob/eb114294efbcd5adb1944c9f3cb5feda}{\texttt{ a few terminals}} began supporting HTML like links, where the link text and destination could be specified separately.

This module allows you to detect if hyperlinks are supported in the current Terminal.

As this is a new development, we anticipate the list of supported Terminals to grow rapidly. Please open an issue or submit a PR as new Terminals implement support.\hypertarget{md_node__modules_2supports-hyperlinks_2readme_autotoc_md27185}{}\doxysection{\texorpdfstring{Install}{Install}}\label{md_node__modules_2supports-hyperlinks_2readme_autotoc_md27185}

Returns an {\ttfamily Object} with a {\ttfamily stdout} and {\ttfamily stderr} property for testing either streams. Each property is a {\ttfamily boolean}, indicating whether or not hyperlinks are supported.\hypertarget{md_node__modules_2supports-hyperlinks_2readme_autotoc_md27188}{}\doxysection{\texorpdfstring{Info}{Info}}\label{md_node__modules_2supports-hyperlinks_2readme_autotoc_md27188}
Obeys the {\ttfamily -\/-\/no-\/hyperlinks}, {\ttfamily -\/-\/hyperlink=always}, and {\ttfamily -\/-\/hyperlink=never} CLI flags.

Can be overridden by the user with the flags {\ttfamily -\/-\/hyperlinks=always} and {\ttfamily -\/-\/no-\/hyperlinks}. For situations where using those flags are not possible, add the environment variable {\ttfamily FORCE\+\_\+\+HYPERLINK=1} to forcefully enable hyperlinks or {\ttfamily FORCE\+\_\+\+HYPERLINK=0} to forcefully disable. The use of {\ttfamily FORCE\+\_\+\+HYPERLINK} overrides all other hyperlink support checks.\hypertarget{md_node__modules_2supports-hyperlinks_2readme_autotoc_md27189}{}\doxysection{\texorpdfstring{Related}{Related}}\label{md_node__modules_2supports-hyperlinks_2readme_autotoc_md27189}
