{\bfseries{Using custom renderers (e.g. {\ttfamily react-\/three-\/fiber}, {\ttfamily react-\/pdf}, {\ttfamily ink})}}\begin{adjustwidth}{1em}{0em}
~\newline


To ensure full support of "{}\+Fast Refresh"{} with components rendered by custom renderers, you should ensure the renderer you\textquotesingle{}re using depends on a recent version of {\ttfamily react-\/reconciler}.

We recommend version {\ttfamily 0.\+25.\+0} or above, but any versions above {\ttfamily 0.\+22.\+0} should work.

If the renderer is not compatible, please file them an issue instead.

\end{adjustwidth}
