 Jest allows you to choose how you want to define focused and skipped tests, with multiple permutations for each\+:


\begin{DoxyItemize}
\item {\bfseries{only \& skip\+:}} {\ttfamily it.\+only}, {\ttfamily test.\+only}, {\ttfamily describe.\+only}, {\ttfamily it.\+skip}, {\ttfamily test.\+skip}, {\ttfamily describe.\+skip}.
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\textquotesingle{}f\textquotesingle{} \& \textquotesingle{}x\textquotesingle{}\+:\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} {\ttfamily fit}, {\ttfamily fdescribe}, {\ttfamily xit}, {\ttfamily xtest}, {\ttfamily xdescribe}.
\end{DoxyItemize}

This rule enforces usages from the {\bfseries{only \& skip}} list.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-test-prefixes_autotoc_md11432}{}\doxysection{\texorpdfstring{Rule details}{Rule details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-test-prefixes_autotoc_md11432}
