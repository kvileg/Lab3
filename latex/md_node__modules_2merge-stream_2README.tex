The argument supplied to {\ttfamily add} can be either a source or an array of sources.\hypertarget{md_node__modules_2merge-stream_2README_autotoc_md18874}{}\doxysubsubsection{\texorpdfstring{$<$tt$>$merged.\+is\+Empty$<$/tt$>$}{<tt>merged.isEmpty</tt>}}\label{md_node__modules_2merge-stream_2README_autotoc_md18874}
A method that tells you if the merged stream is empty.

When a stream is "{}empty"{} (aka. no sources were added), it could not be returned to a gulp task.
