 {\ttfamily to\+Strict\+Equal} not only checks that two objects contain the same data but also that they have the same structure. It is common to expect objects to not only have identical values but also to have identical keys. A stricter equality will catch cases where two objects do not have identical keys.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2prefer-strict-equal_autotoc_md11462}{}\doxysection{\texorpdfstring{Rule details}{Rule details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2prefer-strict-equal_autotoc_md11462}
