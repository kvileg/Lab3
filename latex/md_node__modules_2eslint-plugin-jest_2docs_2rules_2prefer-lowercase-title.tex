Enforce {\ttfamily it}, {\ttfamily test} and {\ttfamily describe} to have descriptions that begin with a lowercase letter. This provides more readable test failures. This rule is not enabled by default.

The following pattern is considered a warning\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{it('Adds\ 1\ +\ 2\ to\ equal\ 3',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ expect(sum(1,\ 2)).toBe(3);}
\DoxyCodeLine{\});}

\end{DoxyCode}


The following pattern is not considered a warning\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{it('adds\ 1\ +\ 2\ to\ equal\ 3',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ expect(sum(1,\ 2)).toBe(3);}
\DoxyCodeLine{\});}

\end{DoxyCode}
