Compare alphanumeric strings the same way a human would, using a natural order algorithm (originally known as the \href{http://davekoelle.com/alphanum.html}{\texttt{ alphanum algorithm}}) where numeric characters are sorted based on their numeric values rather than their ASCII values.

This module exports a function that returns a number indicating whether one string should come before, after, or is the same as another string. It can be used directly with the native \href{https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort}{\texttt{ {\ttfamily .sort()}}} array method.\hypertarget{md_node__modules_2string-natural-compare_2README_autotoc_md26601}{}\doxysubsection{\texorpdfstring{Fast and Robust}{Fast and Robust}}\label{md_node__modules_2string-natural-compare_2README_autotoc_md26601}
This module can compare strings containing any size of number and is heavily tested with a custom \href{https://github.com/nwoltman/string-natural-compare/tree/master/benchmark}{\texttt{ benchmark suite}} to make sure that it is as fast as possible.\hypertarget{md_node__modules_2string-natural-compare_2README_autotoc_md26602}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_node__modules_2string-natural-compare_2README_autotoc_md26602}
