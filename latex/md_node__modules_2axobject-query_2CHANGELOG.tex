Map and Set usages are now replaced with object and array literals.

These are the changes in usage you might need to account for\+:


\begin{DoxyItemize}
\item The module exports are no longer Maps, so you cannot spread them directly into an array to access their items. Use the {\ttfamily entries} method to get access to the items.
\item The {\ttfamily keys} and {\ttfamily values} methods now return arrays, not iterators.
\item There is no {\ttfamily for\+Each} method. One could be added in the future, but at present, it does not exist on the exports.
