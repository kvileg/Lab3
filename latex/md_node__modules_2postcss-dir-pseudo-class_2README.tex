Using \href{https://github.com/csstools/postcss-plugins/tree/main/plugins/postcss-dir-pseudo-class}{\texttt{ Post\+CSS Dir Pseudo Class}} will not impact selector weight, but it will require having at least one {\ttfamily \mbox{[}dir\mbox{]}} attribute in your HTML. If you don’t have {\itshape any} {\ttfamily \mbox{[}dir\mbox{]}} attributes, consider using the following Java\+Script\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ force\ at\ least\ one\ dir\ attribute\ (this\ can\ run\ at\ any\ time)}
\DoxyCodeLine{document.documentElement.dir=document.documentElement.dir||'ltr';}

\end{DoxyCode}

The {\ttfamily dir} option allows you presume a direction in your CSS. By default, this is not specified and you are required to include a direction {\ttfamily \mbox{[}dir\mbox{]}} attribute somewhere in your HTML, preferably on the {\ttfamily html} element.

Here’s an example of using the {\ttfamily dir} option to presume a left-\/to-\/right direction\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{postcssDirPseudoClass(\{\ dir:\ 'ltr'\ \});}

\end{DoxyCode}
