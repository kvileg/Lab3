 \begin{quote}
Safely get a dot-\/notated path within a nested object, with ability to return a default if the full key path does not exist or the value is undefined \end{quote}
\hypertarget{md_node__modules_2dlv_2README_autotoc_md9567}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{md_node__modules_2dlv_2README_autotoc_md9567}
Smallest possible implementation\+: only {\bfseries{130 bytes.}}

You could write this yourself, but then you\textquotesingle{}d have to write \href{https://github.com/developit/dlv/blob/master/test.js}{\texttt{ tests}}.

Supports ES Modules, Common\+JS and globals.\hypertarget{md_node__modules_2dlv_2README_autotoc_md9568}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{md_node__modules_2dlv_2README_autotoc_md9568}
