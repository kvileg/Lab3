Each global is given a value of {\ttfamily true} or {\ttfamily false}. A value of {\ttfamily true} indicates that the variable may be overwritten. A value of {\ttfamily false} indicates that the variable should be considered read-\/only. This information is used by static analysis tools to flag incorrect behavior.
