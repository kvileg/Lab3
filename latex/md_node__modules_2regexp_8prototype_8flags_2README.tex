An ES6 spec-\/compliant {\ttfamily Reg\+Exp.\+prototype.\+flags} shim. Invoke its "{}shim"{} method to shim Reg\+Exp.\+prototype.\+flags if it is unavailable. {\itshape Note}\+: {\ttfamily Reg\+Exp\#flags} requires a true ES5 environment -\/ specifically, one with ES5 getters.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES5-\/supported environment and complies with the \href{http://www.ecma-international.org/ecma-262/6.0/\#sec-get-regexp.prototype.flags}{\texttt{ spec}}.
