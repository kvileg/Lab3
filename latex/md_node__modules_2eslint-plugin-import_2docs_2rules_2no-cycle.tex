Ensures that there is no resolvable path back to this module via its dependencies.

This includes cycles of depth 1 (imported module imports me) to {\ttfamily "{}âˆž"{}} (or {\ttfamily Infinity}), if the \`{}max\+Depth\`{} option is not set.

This rule does {\itshape not} detect imports that resolve directly to the linted module; for that, see \doxysectlink{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-self-import}{\`{}no-\/self-\/import\`{}}{0}.

This rule ignores type-\/only imports in Flow and Type\+Script syntax ({\ttfamily import type} and {\ttfamily import typeof}), which have no runtime effect.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-cycle_autotoc_md10897}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-cycle_autotoc_md10897}

By default, this rule only detects cycles for ES6 imports, but see the \href{./no-unresolved.md\#options}{\texttt{ {\ttfamily no-\/unresolved} options}} as this rule also supports the same {\ttfamily commonjs} and {\ttfamily amd} flags. However, these flags only impact which import types are {\itshape linted}; the import/export infrastructure only registers {\ttfamily import} statements in dependencies, so cycles created by {\ttfamily require} within imported modules may not be detected.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-cycle_autotoc_md10899}{}\doxysubsubsection{\texorpdfstring{$<$tt$>$max\+Depth$<$/tt$>$}{<tt>maxDepth</tt>}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-cycle_autotoc_md10899}

Its value is {\ttfamily false} by default, but can be set to {\ttfamily true} for reducing total project lint time, if needed.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-cycle_autotoc_md10901}{}\doxysubsubsection{\texorpdfstring{$<$tt$>$allow\+Unsafe\+Dynamic\+Cyclic\+Dependency$<$/tt$>$}{<tt>allowUnsafeDynamicCyclicDependency</tt>}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-cycle_autotoc_md10901}
