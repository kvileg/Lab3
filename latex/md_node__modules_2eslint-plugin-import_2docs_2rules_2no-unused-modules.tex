Reports\+:


\begin{DoxyItemize}
\item modules without any exports
\item individual exports not being statically {\ttfamily import}ed or {\ttfamily require}ed from other modules in the same project
\item dynamic imports are supported if argument is a literal string
\end{DoxyItemize}\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10989}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10989}
\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10990}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10990}
In order for this plugin to work, at least one of the options {\ttfamily missing\+Exports} or {\ttfamily unused\+Exports} must be enabled (see "{}\+Options"{} section below). In the future, these options will be enabled by default (see \href{https://github.com/import-js/eslint-plugin-import/issues/1324}{\texttt{ https\+://github.\+com/import-\/js/eslint-\/plugin-\/import/issues/1324}})

\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10991}{}\doxysubsection{\texorpdfstring{Options}{Options}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10991}
This rule takes the following option\+:


\begin{DoxyItemize}
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily missing\+Exports}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+: if {\ttfamily true}, files without any exports are reported (defaults to {\ttfamily false})
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily unused\+Exports}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+: if {\ttfamily true}, exports without any static usage within other modules are reported (defaults to {\ttfamily false})
\item {\ttfamily src}\+: an array with files/paths to be analyzed. It only applies to unused exports. Defaults to {\ttfamily process.\+cwd()}, if not provided
\item {\ttfamily ignore\+Exports}\+: an array with files/paths for which unused exports will not be reported (e.\+g module entry points in a published package)
\end{DoxyItemize}\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10992}{}\doxysubsection{\texorpdfstring{Example for missing exports}{Example for missing exports}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-unused-modules_autotoc_md10992}
