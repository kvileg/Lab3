An ES5 spec-\/compliant {\ttfamily Array.\+prototype.\+reduce} shim/polyfill/replacement that works as far down as ES3.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the \href{https://www.ecma-international.org/ecma-262/5.1/}{\texttt{ spec}}.
