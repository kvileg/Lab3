An Object.\+assign shim. Invoke its "{}shim"{} method to shim Object.\+assign if it is unavailable.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the \href{http://www.ecma-international.org/ecma-262/6.0/\#sec-object.assign}{\texttt{ spec}}. In an ES6 environment, it will also work properly with {\ttfamily Symbol}s.

Takes a minimum of 2 arguments\+: {\ttfamily target} and {\ttfamily source}. Takes a variable sized list of source arguments -\/ at least 1, as many as you want. Throws a Type\+Error if the {\ttfamily target} argument is {\ttfamily null} or {\ttfamily undefined}.
