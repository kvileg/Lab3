This module discards comments from your CSS files; by default, it will remove all regular comments ({\ttfamily /\texorpdfstring{$\ast$}{*} comment \texorpdfstring{$\ast$}{*}/}) and preserve comments marked as important ({\ttfamily /\texorpdfstring{$\ast$}{*}! important \texorpdfstring{$\ast$}{*}/}).

Note that this module does not handle source map comments because they are not available to it; Post\+CSS handles this internally, so if they are removed then you will have to \href{https://github.com/postcss/postcss/blob/master/docs/source-maps.md}{\texttt{ configure source maps in Post\+CSS}}.\hypertarget{md_node__modules_2postcss-discard-comments_2README_autotoc_md20961}{}\doxysection{\texorpdfstring{API}{API}}\label{md_node__modules_2postcss-discard-comments_2README_autotoc_md20961}
