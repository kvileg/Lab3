\hypertarget{md_node__modules_2language-tags_2README_autotoc_md18649}{}\doxysubsection{\texorpdfstring{tags.\+search(description, \mbox{[}all\mbox{]})}{tags.search(description, [all])}}\label{md_node__modules_2language-tags_2README_autotoc_md18649}
Search for tags and subtags by description. Supports either a Reg\+Exp object or a string for {\ttfamily description}. Returns an array of {\ttfamily Subtag} and {\ttfamily Tag} objects or an empty array if no results were found.

Note that {\ttfamily Tag} objects in the results represent \textquotesingle{}grandfathered\textquotesingle{} or \textquotesingle{}redundant\textquotesingle{} tags. These are excluded by default. Set the {\ttfamily all} parameter to {\ttfamily true} to include them.

Search is case-\/insensitive if {\ttfamily description} is a string.\hypertarget{md_node__modules_2language-tags_2README_autotoc_md18650}{}\doxysubsection{\texorpdfstring{tags.\+languages(macrolanguage)}{tags.languages(macrolanguage)}}\label{md_node__modules_2language-tags_2README_autotoc_md18650}
Returns an array of {\ttfamily Subtag} objects representing all the \textquotesingle{}language\textquotesingle{} type subtags belonging to the given \textquotesingle{}macrolanguage\textquotesingle{} type subtag.

Throws an error if {\ttfamily macrolanguage} is not a macrolanguage.

\hypertarget{md_node__modules_2language-tags_2README_autotoc_md18664}{}\doxysubsubsection{\texorpdfstring{subtag.\+format()}{subtag.format()}}\label{md_node__modules_2language-tags_2README_autotoc_md18664}
Return the subtag code formatted according to the case conventions defined in \href{http://tools.ietf.org/html/rfc5646\#section-2.1.1}{\texttt{ RFC 5646 section 2.\+1.\+1}}.


\begin{DoxyItemize}
\item language codes are made lowercase (\textquotesingle{}mn\textquotesingle{} for Mongolian)
\item script codes are made lowercase with the initial letter capitalized (\textquotesingle{}Cyrl\textquotesingle{} for Cyrillic)
\item country codes are capitalized (\textquotesingle{}MN\textquotesingle{} for Mongolia)
\end{DoxyItemize}\hypertarget{md_node__modules_2language-tags_2README_autotoc_md18665}{}\doxysubsection{\texorpdfstring{Tag}{Tag}}\label{md_node__modules_2language-tags_2README_autotoc_md18665}
