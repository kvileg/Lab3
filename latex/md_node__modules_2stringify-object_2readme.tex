Type\+: {\ttfamily Function}~\newline

Expected to return a {\ttfamily string} that transforms the string that resulted from stringifying {\ttfamily obj\mbox{[}prop\mbox{]}}. This can be used to detect special types of objects that need to be stringified in a particular way. The {\ttfamily transform} function might return an alternate string in this case, otherwise returning the {\ttfamily original\+Result}.
