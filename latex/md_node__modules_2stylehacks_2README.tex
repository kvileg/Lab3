\hypertarget{md_node__modules_2stylehacks_2README_autotoc_md26956}{}\doxysubsection{\texorpdfstring{$<$tt$>$stylehacks.\+detect(node)$<$/tt$>$}{<tt>stylehacks.detect(node)</tt>}}\label{md_node__modules_2stylehacks_2README_autotoc_md26956}
Type\+: {\ttfamily function} ~\newline
 Returns\+: {\ttfamily boolean}

This method will take any Post\+CSS {\itshape node}, run applicable plugins depending on its type, then will return a boolean depending on whether it found any of the supported hacks. For example, if the {\ttfamily decl} node found below is passed to the {\ttfamily detect} function, it will return {\ttfamily true}. But if the {\ttfamily rule} node is passed, it will return {\ttfamily false} instead.
