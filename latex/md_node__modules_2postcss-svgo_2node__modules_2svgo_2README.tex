\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+SVG O\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}ptimizer is a Node.\+js-\/based tool for optimizing SVG vector graphics files.\hypertarget{md_node__modules_2postcss-svgo_2node__modules_2svgo_2README_autotoc_md22141}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{md_node__modules_2postcss-svgo_2node__modules_2svgo_2README_autotoc_md22141}
SVG files, especially those exported from various editors, usually contain a lot of redundant and useless information. This can include editor metadata, comments, hidden elements, default or non-\/optimal values and other stuff that can be safely removed or converted without affecting the SVG rendering result.\hypertarget{md_node__modules_2postcss-svgo_2node__modules_2svgo_2README_autotoc_md22142}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{md_node__modules_2postcss-svgo_2node__modules_2svgo_2README_autotoc_md22142}
