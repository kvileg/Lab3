Some special networking features can optionally use a Flash component. Building the output SWF file requires the \href{https://flex.apache.org/}{\texttt{ Flex SDK}}. A pre-\/built component is included\+: {\ttfamily swf/\+Socket\+Pool.\+swf}.

Building the output SWF requires the {\ttfamily mxmlc} tool from the \href{https://flex.apache.org/}{\texttt{ Flex SDK}}. If that tools is already installed then look in the {\ttfamily package.\+json} file for the commands to rebuild it. If you need the SDK installed, there is a npm module that installs it\+: \begin{DoxyVerb}npm install
\end{DoxyVerb}
 To build a regular component\+: \begin{DoxyVerb}npm run build
\end{DoxyVerb}
 Additional debug support can be built in with the following\+: \begin{DoxyVerb}npm run build-debug
\end{DoxyVerb}
