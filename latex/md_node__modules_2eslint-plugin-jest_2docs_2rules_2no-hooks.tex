 Jest provides global functions for setup and teardown tasks, which are called before/after each test case and each test suite. The use of these hooks promotes shared state between tests.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-hooks_autotoc_md11399}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-hooks_autotoc_md11399}
This rule reports for the following function calls\+:


\begin{DoxyItemize}
\item {\ttfamily before\+All}
\item {\ttfamily before\+Each}
\item {\ttfamily after\+All}
\item {\ttfamily after\+Each}
\end{DoxyItemize}

\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-hooks_autotoc_md11401}{}\doxysubsection{\texorpdfstring{$<$tt$>$allow$<$/tt$>$}{<tt>allow</tt>}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-hooks_autotoc_md11401}
This array option controls which Jest hooks are checked by this rule. There are four possible values\+:


\begin{DoxyItemize}
\item {\ttfamily "{}before\+All"{}}
\item {\ttfamily "{}before\+Each"{}}
\item {\ttfamily "{}after\+All"{}}
\item {\ttfamily "{}after\+Each"{}}
\end{DoxyItemize}

By default, none of these options are enabled (the equivalent of {\ttfamily \{ "{}allow"{}\+: \mbox{[}\mbox{]} \}}).
