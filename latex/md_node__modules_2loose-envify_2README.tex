Fast (and loose) selective {\ttfamily process.\+env} replacer using \href{https://github.com/lydell/js-tokens}{\texttt{ js-\/tokens}} instead of an AST. Works just like \href{https://github.com/hughsk/envify}{\texttt{ envify}} but much faster.\hypertarget{md_node__modules_2loose-envify_2README_autotoc_md18761}{}\doxysection{\texorpdfstring{Gotchas}{Gotchas}}\label{md_node__modules_2loose-envify_2README_autotoc_md18761}

\begin{DoxyItemize}
\item Doesn\textquotesingle{}t handle broken syntax.
\item Doesn\textquotesingle{}t look inside embedded expressions in template strings.
\begin{DoxyItemize}
\item {\bfseries{this won\textquotesingle{}t work\+:}} 
\begin{DoxyCode}{0}

\end{DoxyCode}

\end{DoxyItemize}
\item Doesn\textquotesingle{}t replace oddly-\/spaced or oddly-\/commented expressions.
\begin{DoxyItemize}
\item {\bfseries{this won\textquotesingle{}t work\+:}} 
\begin{DoxyCode}{0}

\end{DoxyCode}

\end{DoxyItemize}
