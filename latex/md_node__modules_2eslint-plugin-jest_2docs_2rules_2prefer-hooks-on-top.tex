 While hooks can be setup anywhere in a test file, they are always called in a specific order which means it can be confusing if they\textquotesingle{}re intermixed with test cases.

This rule helps to ensure that hooks are always defined before test cases.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2prefer-hooks-on-top_autotoc_md11451}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2prefer-hooks-on-top_autotoc_md11451}
