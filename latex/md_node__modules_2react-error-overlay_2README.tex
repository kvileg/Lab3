When developing within this package, make sure you run {\ttfamily npm start} (or {\ttfamily yarn start}) so that the files are compiled as you work. This is run in watch mode by default.

If you would like to build this for production, run {\ttfamily npm run build\+:prod} (or {\ttfamily yarn build\+:prod}).~\newline
