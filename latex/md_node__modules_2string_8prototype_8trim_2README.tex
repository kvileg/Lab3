An ES5 spec-\/compliant {\ttfamily String.\+prototype.\+trim} shim. Invoke its "{}shim"{} method to shim {\ttfamily String.\+prototype.\+trim} if it is unavailable.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the spec (both \href{https://262.ecma-international.org/5.1/\#sec-15.5.4.20}{\texttt{ ES5}} and \href{https://tc39.es/ecma262/\#sec-string.prototype.trim}{\texttt{ current}}).
