Use this rule to prevent importing the submodules of other modules.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-internal-modules_autotoc_md10936}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-internal-modules_autotoc_md10936}
This rule has two mutally exclusive options that are arrays of \href{https://github.com/isaacs/node-glob\#glob-primer}{\texttt{ minimatch/glob patterns}} patterns\+:


\begin{DoxyItemize}
\item {\ttfamily allow} that include paths and import statements that can be imported with reaching.
\item {\ttfamily forbid} that exclude paths and import statements that can be imported with reaching.
\end{DoxyItemize}\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-internal-modules_autotoc_md10937}{}\doxysubsection{\texorpdfstring{Examples}{Examples}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-internal-modules_autotoc_md10937}
