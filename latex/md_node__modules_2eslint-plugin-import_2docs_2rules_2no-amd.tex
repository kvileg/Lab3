Reports {\ttfamily require(\mbox{[}array\mbox{]}, ...)} and {\ttfamily define(\mbox{[}array\mbox{]}, ...)} function calls at the module scope. Will not report if !=2 arguments, or first argument is not a literal array.

Intended for temporary use when migrating to pure ES6 modules.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-amd_autotoc_md10879}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2no-amd_autotoc_md10879}

\DoxyCodeLine{define(["{}a"{},\ "{}b"{}],\ function\ (a,\ b)\ \{\ /*\ ...\ */\ \})}
\DoxyCodeLine{}
\DoxyCodeLine{require(["{}b"{},\ "{}c"{}],\ function\ (b,\ c)\ \{\ /*\ ...\ */\ \})}

\end{DoxyCode}
