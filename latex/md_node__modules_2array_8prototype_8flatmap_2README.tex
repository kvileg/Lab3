An ES2019 spec-\/compliant {\ttfamily Array.\+prototype.\+flat\+Map} shim/polyfill/replacement that works as far down as ES3.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the \href{https://tc39.es/ecma262/\#sec-array.prototype.flatmap}{\texttt{ spec}}.
