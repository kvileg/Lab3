 {\ttfamily jest-\/docblock} is a package that can extract and parse a specially-\/formatted comment called a "{}docblock"{} at the top of a file.

A docblock looks like this\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{/**}
\DoxyCodeLine{\ *\ Stuff\ goes\ here!}
\DoxyCodeLine{\ */}

\end{DoxyCode}


Docblocks can contain pragmas, which are words prefixed by {\ttfamily @}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{/**}
\DoxyCodeLine{\ *\ Pragma\ incoming!}
\DoxyCodeLine{\ *}
\DoxyCodeLine{\ *\ @flow}
\DoxyCodeLine{\ */}

\end{DoxyCode}


Pragmas can also take arguments\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{/**}
\DoxyCodeLine{\ *\ Check\ this\ out:}
\DoxyCodeLine{\ *}
\DoxyCodeLine{\ *\ @myPragma\ it\ is\ so\ cool}
\DoxyCodeLine{\ */}

\end{DoxyCode}


{\ttfamily jest-\/docblock} can\+:


\begin{DoxyItemize}
\item extract the docblock from some code as a string
\item parse a docblock string\textquotesingle{}s pragmas into an object
\item print an object and some comments back to a string
