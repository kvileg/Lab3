\href{https://github.com/postcss/postcss}{\texttt{ Post\+CSS}} plugin to transform {\ttfamily clamp()} to combination of {\ttfamily min/max}.

This plugin transform this css\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{.foo\ \{}
\DoxyCodeLine{\ \ width:\ clamp(10px,\ 4em,\ 80px);}
\DoxyCodeLine{\}}

\end{DoxyCode}


into this\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{.foo\ \{}
\DoxyCodeLine{\ \ width:\ max(10px,\ min(4em,\ 80px));}
\DoxyCodeLine{\}}

\end{DoxyCode}


Or with enabled options {\ttfamily precalculate}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{.foo\ \{}
\DoxyCodeLine{\ \ width:\ clamp(10em,\ 4px,\ 10px);}
\DoxyCodeLine{\}}
\DoxyCodeLine{}
\DoxyCodeLine{/*\ becomes\ */}
\DoxyCodeLine{}
\DoxyCodeLine{.foo\ \{}
\DoxyCodeLine{\ \ width:\ max(10em,\ 14px);}
\DoxyCodeLine{\}}

\end{DoxyCode}

The {\ttfamily precalculate} option determines whether values with the same unit should be precalculated. By default, these are not precalculation.


\begin{DoxyCode}{0}
\DoxyCodeLine{postcssColorHexAlpha(\{}
\DoxyCodeLine{\ \ precalculate:\ true}
\DoxyCodeLine{\});}

\end{DoxyCode}


The second and third value has the same unit ({\ttfamily px})\+:


\begin{DoxyCode}{0}

\DoxyCodeLine{\ \ width:\ clamp(10em,\ 4px,\ 10px);}
\DoxyCodeLine{\}}
\DoxyCodeLine{}
\DoxyCodeLine{/*\ becomes\ */}
\DoxyCodeLine{}
\DoxyCodeLine{.foo\ \{}
\DoxyCodeLine{\ \ width:\ max(10em,\ 14px);}
\DoxyCodeLine{\}}
