\href{https://html.spec.whatwg.org/multipage/webappapis.html\#atob:dom-windowbase64-btoa-3}{\texttt{ Per the spec}}, {\ttfamily btoa} will accept strings "{}containing only characters in the range \`{}\+U+0000\`{} to \`{}\+U+00\+FF\`{}."{} If passed a string with characters above {\ttfamily U+00\+FF}, {\ttfamily btoa} will return {\ttfamily null}. If {\ttfamily atob} is passed a string that is not base64-\/valid, it will also return {\ttfamily null}.
