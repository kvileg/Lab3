\chapter{symbol-\/tree}
\hypertarget{md_node__modules_2symbol-tree_2README}{}\label{md_node__modules_2symbol-tree_2README}\index{symbol-\/tree@{symbol-\/tree}}
\href{https://travis-ci.org/jsdom/js-symbol-tree}{\texttt{ }} \href{https://coveralls.io/github/jsdom/js-symbol-tree?branch=master}{\texttt{ }}

Turn any collection of objects into its own efficient tree or linked list using {\ttfamily Symbol}.

This library has been designed to provide an efficient backing data structure for DOM trees. You can also use this library as an efficient linked list. Any meta data is stored on your objects directly, which ensures any kind of insertion or deletion is performed in constant time. Because an ES6 {\ttfamily Symbol} is used, the meta data does not interfere with your object in any way.

Node.\+js 4+, io.\+js and modern browsers are supported.\hypertarget{md_node__modules_2symbol-tree_2README_autotoc_md27328}{}\doxysection{\texorpdfstring{Example}{Example}}\label{md_node__modules_2symbol-tree_2README_autotoc_md27328}
