\Hypertarget{md_node__modules_2xml-name-validator_2README_autotoc_md30564}%
 This package simply tells you whether or not a string matches the \href{http://www.w3.org/TR/xml/\#NT-Name}{\texttt{ {\ttfamily Name}}} or \href{http://www.w3.org/TR/xml-names/\#NT-QName}{\texttt{ {\ttfamily QName}}} productions in the XML Namespaces specification. We use it for implementing the \href{https://dom.spec.whatwg.org/\#validate}{\texttt{ validate}} algorithm in jsdom, but you can use it for whatever you want.\hypertarget{md_node__modules_2xml-name-validator_2README_autotoc_md30565}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md_node__modules_2xml-name-validator_2README_autotoc_md30565}
This package\textquotesingle{}s main module\textquotesingle{}s default export takes a string and will return an object of the form {\ttfamily \{ success, error \}}, where {\ttfamily success} is a boolean and if it is {\ttfamily false}, then {\ttfamily error} is a string containing some hint as to where the match went wrong.
