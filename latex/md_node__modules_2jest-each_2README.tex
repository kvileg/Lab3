 \doxysection*{jest-\/each}

 Jest Parameterised Testing 

\DoxyHorRuler{0}


\href{https://www.npmjs.com/package/jest-each}{\texttt{ }} \href{http://npm-stat.com/charts.html?package=jest-each&from=2017-03-21}{\texttt{ }} \href{https://github.com/facebook/jest/blob/main/LICENSE}{\texttt{ }}

A parameterised testing library for \href{https://jestjs.io/}{\texttt{ Jest}} inspired by \href{https://github.com/ryym/mocha-each}{\texttt{ mocha-\/each}}.

jest-\/each allows you to provide multiple arguments to your {\ttfamily test}/{\ttfamily describe} which results in the test/suite being run once per row of parameters.\hypertarget{md_node__modules_2jest-each_2README_autotoc_md17063}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{md_node__modules_2jest-each_2README_autotoc_md17063}
