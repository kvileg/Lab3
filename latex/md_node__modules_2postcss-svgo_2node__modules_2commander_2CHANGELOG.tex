The biggest change is the parsed option values. Previously the options were stored by default as properties on the command object, and now the options are stored separately.

If you wish to restore the old behaviour and get running quickly you can call {\ttfamily .store\+Options\+As\+Properties()}. To allow you to move to the new code patterns incrementally, the action handler will be passed the command {\itshape twice}, to match the new "{}options"{} and "{}command"{} parameters (see below).

{\bfseries{program options}}

Use the {\ttfamily .opts()} method to access the options. This is available on any command but is used most with the program.


\begin{DoxyCode}{0}
\DoxyCodeLine{program.option('-\/d,\ -\/-\/debug');}
\DoxyCodeLine{program.parse();}
\DoxyCodeLine{//\ Old\ code\ before\ Commander\ 7}
\DoxyCodeLine{if\ (program.debug)\ console.log(`Program\ name\ is\ \$\{program.name()\}`);}

\end{DoxyCode}



\begin{DoxyCode}{0}
\DoxyCodeLine{//\ New\ code}
\DoxyCodeLine{const\ options\ =\ program.opts();}
\DoxyCodeLine{if\ (options.debug)\ console.log(`Program\ name\ is\ \$\{program.name()\}`);}

\end{DoxyCode}


{\bfseries{action handler}}

The action handler gets passed a parameter for each command-\/argument you declared. Previously by default the next parameter was the command object with the options as properties. Now the next two parameters are instead the options and the command. If you only accessed the options there may be no code changes required.


\begin{DoxyCode}{0}
\DoxyCodeLine{program}
\DoxyCodeLine{\ \ .command('compress\ <filename>')}
\DoxyCodeLine{\ \ .option('-\/t,\ -\/-\/trace')}
\DoxyCodeLine{\ \ //\ Old\ code\ before\ Commander\ 7}
\DoxyCodeLine{\ \ .action((filename,\ cmd))\ =>\ \{}
\DoxyCodeLine{\ \ \ \ if\ (cmd.trace)\ console.log(`Command\ name\ is\ \$\{cmd.name()\}`);}
\DoxyCodeLine{\ \ \});}

\end{DoxyCode}



\begin{DoxyCode}{0}
\DoxyCodeLine{//\ New\ code}
\DoxyCodeLine{.action((filename,\ options,\ command))\ =>\ \{}
\DoxyCodeLine{\ \ if\ (options.trace)\ console.log(`Command\ name\ is\ \$\{command.name()\}`);}
\DoxyCodeLine{\});}

\end{DoxyCode}


If you already set {\ttfamily .store\+Options\+As\+Properties(false)} you may still need to adjust your code.


\begin{DoxyCode}{0}
\DoxyCodeLine{program}
\DoxyCodeLine{\ \ .command('compress\ <filename>')}
\DoxyCodeLine{\ \ .storeOptionsAsProperties(false)}
\DoxyCodeLine{\ \ .option('-\/t,\ -\/-\/trace')}
\DoxyCodeLine{\ \ //\ Old\ code\ before\ Commander\ 7}
\DoxyCodeLine{\ \ .action((filename,\ command))\ =>\ \{}
\DoxyCodeLine{\ \ \ \ if\ (command.opts().trace)\ console.log(`Command\ name\ is\ \$\{command.name()\}`);}
\DoxyCodeLine{\ \ \});}

\end{DoxyCode}



\begin{DoxyCode}{0}
\DoxyCodeLine{//\ New\ code}
\DoxyCodeLine{.action((filename,\ options,\ command))\ =>\ \{}
\DoxyCodeLine{\ \ \ if\ (command.opts().trace)\ console.log(`Command\ name\ is\ \$\{command.name()\}`);}
\DoxyCodeLine{\});}

\end{DoxyCode}
