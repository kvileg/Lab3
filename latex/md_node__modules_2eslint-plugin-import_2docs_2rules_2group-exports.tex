Reports when named exports are not grouped together in a single {\ttfamily export} declaration or when multiple assignments to Common\+JS {\ttfamily module.\+exports} or {\ttfamily exports} object are present in a single file.

{\bfseries{Rationale\+:}} An {\ttfamily export} declaration or {\ttfamily module.\+exports} assignment can appear anywhere in the code. By requiring a single export declaration all your exports will remain at one place, making it easier to see what exports a module provides.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2group-exports_autotoc_md10845}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2group-exports_autotoc_md10845}
