{\bfseries{Esprima}} (\href{http://esprima.org}{\texttt{ esprima.\+org}}, BSD license) is a high performance, standard-\/compliant \href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ ECMAScript}} parser written in ECMAScript (also popularly known as \href{http://en.wikipedia.org/wiki/JavaScript>JavaScript}{\texttt{ Java\+Script}}). Esprima is created and maintained by \href{http://twitter.com/ariyahidayat}{\texttt{ Ariya Hidayat}}, with the help of \href{https://github.com/ariya/esprima/contributors}{\texttt{ many contributors}}.\hypertarget{md_node__modules_2jsonpath_2node__modules_2esprima_2README_autotoc_md18441}{}\doxysubsubsection{\texorpdfstring{Features}{Features}}\label{md_node__modules_2jsonpath_2node__modules_2esprima_2README_autotoc_md18441}

\begin{DoxyItemize}
\item Full support for ECMAScript 5.\+1 (\href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ ECMA-\/262}})
\item Sensible \href{http://esprima.org/doc/index.html\#ast}{\texttt{ syntax tree format}} compatible with Mozilla \href{https://developer.mozilla.org/en/SpiderMonkey/Parser_API}{\texttt{ Parser AST}}
\item Optional tracking of syntax node location (index-\/based and line-\/column)
\item Heavily tested (\texorpdfstring{$>$}{>} 650 \href{http://esprima.org/test/}{\texttt{ unit tests}} with \href{http://esprima.org/test/coverage.html}{\texttt{ full code coverage}})
\item \href{http://esprima.org/doc/es6.html}{\texttt{ Partial support}} for ECMAScript 6
\end{DoxyItemize}

Esprima serves as a {\bfseries{building block}} for some Java\+Script language tools, from \href{http://esprima.org/demo/functiontrace.html}{\texttt{ code instrumentation}} to \href{http://esprima.org/demo/autocomplete.html}{\texttt{ editor autocompletion}}.

Esprima runs on many popular web browsers, as well as other ECMAScript platforms such as \href{http://www.mozilla.org/rhino}{\texttt{ Rhino}}, \href{http://openjdk.java.net/projects/nashorn/}{\texttt{ Nashorn}}, and \href{https://npmjs.org/package/esprima}{\texttt{ Node.\+js}}.

For more information, check the web site \href{http://esprima.org}{\texttt{ esprima.\+org}}.
