This function takes a string {\ttfamily property} that is a canonical/unaliased Unicode property name, and a string {\ttfamily value}. It attemps to match {\ttfamily value} to a canonical Unicode property value for the given property. If there’s a match, it returns the canonical property value. Otherwise, it throws an exception.
