 {\itshape unicode-\/property-\/aliases-\/ecmascript} offers Unicode property alias mappings in an easy-\/to-\/consume Java\+Script format. It only contains the Unicode property names that are supported in \href{https://github.com/tc39/proposal-regexp-unicode-property-escapes}{\texttt{ ECMAScript Reg\+Exp property escapes}}.

It’s based on Unicode’s {\ttfamily Property\+Aliases.\+txt}.\hypertarget{md_node__modules_2unicode-property-aliases-ecmascript_2README_autotoc_md28977}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_node__modules_2unicode-property-aliases-ecmascript_2README_autotoc_md28977}

\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ npm\ install\ unicode-\/property-\/aliases-\/ecmascript}
