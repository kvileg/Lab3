\begin{quote}
Modular standard library for Java\+Script. Includes polyfills for \href{https://github.com/zloirock/core-js\#ecmascript}{\texttt{ ECMAScript up to 2023}}\+: \href{https://github.com/zloirock/core-js\#ecmascript-promise}{\texttt{ promises}}, \href{https://github.com/zloirock/core-js\#ecmascript-symbol}{\texttt{ symbols}}, \href{https://github.com/zloirock/core-js\#ecmascript-collections}{\texttt{ collections}}, iterators, \href{https://github.com/zloirock/core-js\#ecmascript-typed-arrays}{\texttt{ typed arrays}}, many other features, \href{https://github.com/zloirock/core-js\#ecmascript-proposals}{\texttt{ ECMAScript proposals}}, some cross-\/platform WHATWG / W3C features and proposals like \href{https://github.com/zloirock/core-js\#url-and-urlsearchparams}{\texttt{ {\ttfamily URL}}}. You can load only required features or use it without global namespace pollution. \end{quote}
