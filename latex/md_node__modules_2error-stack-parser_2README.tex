Simple, cross-\/browser \href{https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Error}{\texttt{ Error}} parser. This library parses and extracts function names, URLs, line numbers, and column numbers from the given Error\textquotesingle{}s {\ttfamily stack} as an Array of \href{http://git.io/stackframe}{\texttt{ Stack\+Frame}}s.

Once you have parsed out Stack\+Frames, you can do much more interesting things. See \href{http://git.io/stacktrace-gps}{\texttt{ stacktrace-\/gps}}.

Note that in IE9 and earlier, {\ttfamily Error} objects don\textquotesingle{}t have enough information to extract much of anything.
