 This package includes polyfills for various browsers. It includes minimum requirements and commonly used language features used by \href{https://github.com/facebook/create-react-app}{\texttt{ Create React App}} projects.\hypertarget{md_node__modules_2react-app-polyfill_2README_autotoc_md22808}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{md_node__modules_2react-app-polyfill_2README_autotoc_md22808}
First, install the package using Yarn or npm\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install\ react-\/app-\/polyfill}

\end{DoxyCode}


or


\begin{DoxyCode}{0}
\DoxyCodeLine{yarn\ add\ react-\/app-\/polyfill}

\end{DoxyCode}
\hypertarget{md_node__modules_2react-app-polyfill_2README_autotoc_md22809}{}\doxysection{\texorpdfstring{Supporting Internet Explorer}{Supporting Internet Explorer}}\label{md_node__modules_2react-app-polyfill_2README_autotoc_md22809}
You can import the entry point for the minimal version you intend to support to ensure that the minimum language features are present that are required to use Create React App. For example, if you import the IE9 entry point, this will include IE10 and IE11 support.

These modules ensure the following language features are present\+:


\begin{DoxyEnumerate}
\item {\ttfamily Promise} (for {\ttfamily async} / {\ttfamily await} support)
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item {\ttfamily window.\+fetch} (a Promise-\/based way to make web requests in the browser)
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item {\ttfamily Object.\+assign} (a helper required for Object Spread, i.\+e. {\ttfamily \{ ...a, ...b \}})
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item {\ttfamily Symbol} (a built-\/in object used by {\ttfamily for...of} syntax and friends)
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item {\ttfamily Array.\+from} (a built-\/in static method used by array spread, i.\+e. {\ttfamily \mbox{[}...arr\mbox{]}})
\end{DoxyEnumerate}

{\itshape If you need more features, see the Polyfilling other language features section below.}\hypertarget{md_node__modules_2react-app-polyfill_2README_autotoc_md22810}{}\doxysubsubsection{\texorpdfstring{Internet Explorer 9}{Internet Explorer 9}}\label{md_node__modules_2react-app-polyfill_2README_autotoc_md22810}

You can also polyfill stable language features not available in your target browsers. If you\textquotesingle{}re using this in Create React App, it will automatically use the {\ttfamily browserslist} you\textquotesingle{}ve defined to only include polyfills needed by your target browsers when importing the {\ttfamily stable} polyfill. {\bfseries{Make sure to follow the Internet Explorer steps above if you need to support Internet Explorer in your application}}.


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ This\ must\ be\ the\ first\ line\ in\ src/index.js}
\DoxyCodeLine{import\ 'react-\/app-\/polyfill/stable';}
\DoxyCodeLine{}
\DoxyCodeLine{//\ ...}

\end{DoxyCode}
