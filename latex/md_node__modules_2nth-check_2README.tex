 Parses and compiles CSS nth-\/checks to highly optimized functions.\hypertarget{md_node__modules_2nth-check_2README_autotoc_md19736}{}\doxysubsection{\texorpdfstring{About}{About}}\label{md_node__modules_2nth-check_2README_autotoc_md19736}
This module can be used to parse \& compile nth-\/checks, as they are found in CSS 3\textquotesingle{}s {\ttfamily nth-\/child()} and {\ttfamily nth-\/last-\/of-\/type()}. It can be used to check if a given index matches a given nth-\/rule, or to generate a sequence of indices matching a given nth-\/rule.

{\ttfamily nth-\/check} focusses on speed, providing optimized functions for different kinds of nth-\/child formulas, while still following the \href{http://www.w3.org/TR/css3-selectors/\#nth-child-pseudo}{\texttt{ spec}}.\hypertarget{md_node__modules_2nth-check_2README_autotoc_md19737}{}\doxysubsection{\texorpdfstring{API}{API}}\label{md_node__modules_2nth-check_2README_autotoc_md19737}

\hypertarget{md_node__modules_2nth-check_2README_autotoc_md19738}{}\doxyparagraph{\texorpdfstring{$<$tt$>$nth\+Check(formula)$<$/tt$>$}{<tt>nthCheck(formula)</tt>}}\label{md_node__modules_2nth-check_2README_autotoc_md19738}
Parses and compiles a formula to a highly optimized function. Combination of {\ttfamily parse} and {\ttfamily compile}.

If the formula doesn\textquotesingle{}t match any elements, it returns \href{https://github.com/fb55/boolbase}{\texttt{ {\ttfamily boolbase}}}\textquotesingle{}s {\ttfamily false\+Func}. Otherwise, a function accepting an {\itshape index} is returned, which returns whether or not the passed {\itshape index} matches the formula.

{\bfseries{Note}}\+: The nth-\/rule starts counting at {\ttfamily 1}, the returned function at {\ttfamily 0}.
