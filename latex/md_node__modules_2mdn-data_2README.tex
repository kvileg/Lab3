Maintained by the \href{https://wiki.mozilla.org/MDN}{\texttt{ MDN team at Mozilla}}.

This repository contains general data for Web technologies.

This data is used in MDN documentation, to build \href{https://developer.mozilla.org/en-US/docs/Web/CSS/background}{\texttt{ information boxes}} or \href{https://developer.mozilla.org/en-US/docs/Web/API/Window}{\texttt{ sidebar navigation}}. External tools have started to make use of this data as well. For example, the \href{https://github.com/csstree/csstree/}{\texttt{ CSSTree}} CSS parser.
