There are two options available\+: {\ttfamily single} and {\ttfamily any}. By default, if you do not specify the option, rule will assume it is {\ttfamily single}.\hypertarget{md_node__modules_2eslint-plugin-import_2docs_2rules_2prefer-default-export_autotoc_md11024}{}\doxysubsubsection{\texorpdfstring{single}{single}}\label{md_node__modules_2eslint-plugin-import_2docs_2rules_2prefer-default-export_autotoc_md11024}
{\bfseries{Definition}}\+: When there is only a single export from a module, prefer using default export over named export.
