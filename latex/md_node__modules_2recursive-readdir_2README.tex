Recursively list all files in a directory and its subdirectories. It does not list the directories themselves.

Because it uses fs.\+readdir, which calls \href{http://linux.die.net/man/3/readdir}{\texttt{ readdir}} under the hood on OS X and Linux, the order of files inside directories is \href{http://stackoverflow.com/questions/8977441/does-readdir-guarantee-an-order}{\texttt{ not guaranteed}}.\hypertarget{md_node__modules_2recursive-readdir_2README_autotoc_md23047}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_node__modules_2recursive-readdir_2README_autotoc_md23047}
\begin{DoxyVerb}npm install recursive-readdir
\end{DoxyVerb}
