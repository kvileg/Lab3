\chapter{Stable}
\hypertarget{md_node__modules_2stable_2README}{}\label{md_node__modules_2stable_2README}\index{Stable@{Stable}}

 A stable array sort, because {\ttfamily Array\#sort()} is not guaranteed stable.

MIT licensed.

\href{http://travis-ci.org/Two-Screen/stable}{\texttt{ }}

\href{http://ci.testling.com/Two-Screen/stable}{\texttt{ }}\hypertarget{md_node__modules_2stable_2README_autotoc_md26334}{}\doxysubsubsection{\texorpdfstring{From npm}{From npm}}\label{md_node__modules_2stable_2README_autotoc_md26334}
Install with\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install\ stable}

\end{DoxyCode}


Then use it in Node.\+js or some other Common\+JS environment as\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ stable\ =\ require('stable')}

\end{DoxyCode}
\hypertarget{md_node__modules_2stable_2README_autotoc_md26335}{}\doxysubsubsection{\texorpdfstring{From the browser}{From the browser}}\label{md_node__modules_2stable_2README_autotoc_md26335}
Include \href{https://raw.github.com/Two-Screen/stable/master/stable.js}{\texttt{ {\ttfamily stable.\+js}}} or the minified version \href{https://raw.github.com/Two-Screen/stable/master/stable.min.js}{\texttt{ {\ttfamily stable.\+min.\+js}}} in your page, then call {\ttfamily stable()}.\hypertarget{md_node__modules_2stable_2README_autotoc_md26336}{}\doxysubsubsection{\texorpdfstring{Usage}{Usage}}\label{md_node__modules_2stable_2README_autotoc_md26336}
The default sort is, as with {\ttfamily Array\#sort}, lexicographical\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{stable(['foo',\ 'bar',\ 'baz'])\ \ //\ =>\ ['bar',\ 'baz',\ 'foo']}
\DoxyCodeLine{stable([10,\ 1,\ 5])\ \ \ \ \ \ \ \ \ \ \ \ \ //\ =>\ [1,\ 10,\ 5]}

\end{DoxyCode}


Unlike {\ttfamily Array\#sort}, the default sort is {\bfseries{NOT}} in-\/place. To do an in-\/place sort, use {\ttfamily stable.\+inplace}, which otherwise works the same\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ arr\ =\ [10,\ 1,\ 5]}
\DoxyCodeLine{stable(arr)\ ===\ arr\ \ \ \ \ \ \ \ \ \ //\ =>\ false}
\DoxyCodeLine{stable.inplace(arr)\ ===\ arr\ \ //\ =>\ true}

\end{DoxyCode}


A comparator function can be specified\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ Regular\ sort()\ compatible\ comparator,\ that\ returns\ a\ number.}
\DoxyCodeLine{//\ This\ demonstrates\ the\ default\ behavior.}
\DoxyCodeLine{const\ lexCmp\ =\ (a,\ b)\ =>\ String(a).localeCompare(b)}
\DoxyCodeLine{stable(['foo',\ 'bar',\ 'baz'],\ lexCmp)\ \ //\ =>\ ['bar',\ 'baz',\ 'foo']}
\DoxyCodeLine{}
\DoxyCodeLine{//\ Boolean\ comparator.\ Sorts\ \`{}b`\ before\ \`{}a`\ if\ true.}
\DoxyCodeLine{//\ This\ demonstrates\ a\ simple\ way\ to\ sort\ numerically.}
\DoxyCodeLine{const\ greaterThan\ =\ (a,\ b)\ =>\ a\ >\ b}
\DoxyCodeLine{stable([10,\ 1,\ 5],\ greaterThan)\ \ //\ =>\ [1,\ 5,\ 10]}

\end{DoxyCode}
