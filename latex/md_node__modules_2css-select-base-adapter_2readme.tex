An adapter for {\ttfamily css-\/select} requires the following functions to be implemented\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{isTag,\ existsOne,\ getAttributeValue,\ getChildren,\ getName,\ getParent,}
\DoxyCodeLine{getSiblings,\ getText,\ hasAttrib,\ removeSubsets,\ findAll,\ findOne}

\end{DoxyCode}


You can pass this module a more minimal implementation and it will return a full adapter that fills in any missing functions, provided that you implement at least\+: ~\newline



\begin{DoxyCode}{0}
\DoxyCodeLine{isTag,\ getAttributeValue,\ getChildren,\ getName,\ getParent,\ getText}

\end{DoxyCode}


If you provide any of the other methods required of an adapter, the base adapter will use your implementation instead of its own.
