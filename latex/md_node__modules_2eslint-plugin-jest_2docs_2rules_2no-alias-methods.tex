\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-alias-methods_autotoc_md11370}%
\Hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-alias-methods_autotoc_md11370}%
 Several Jest methods have alias names, such as {\ttfamily to\+Throw} having the alias of {\ttfamily to\+Throw\+Error}. This rule ensures that only the canonical name as used in the Jest documentation is used in the code. This makes it easier to search for all occurrences of the method within code, and it ensures consistency among the method names used.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-alias-methods_autotoc_md11371}{}\doxysection{\texorpdfstring{Rule details}{Rule details}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-alias-methods_autotoc_md11371}
This rule triggers a warning if the alias name, rather than the canonical name, of a method is used.\hypertarget{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-alias-methods_autotoc_md11372}{}\doxysubsection{\texorpdfstring{Default configuration}{Default configuration}}\label{md_node__modules_2eslint-plugin-jest_2docs_2rules_2no-alias-methods_autotoc_md11372}
The following patterns are considered warnings\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{expect(a).toBeCalled();}
\DoxyCodeLine{expect(a).toBeCalledTimes();}
\DoxyCodeLine{expect(a).toBeCalledWith();}
\DoxyCodeLine{expect(a).lastCalledWith();}
\DoxyCodeLine{expect(a).nthCalledWith();}
\DoxyCodeLine{expect(a).toReturn();}
\DoxyCodeLine{expect(a).toReturnTimes();}
\DoxyCodeLine{expect(a).toReturnWith();}
\DoxyCodeLine{expect(a).lastReturnedWith();}
\DoxyCodeLine{expect(a).nthReturnedWith();}
\DoxyCodeLine{expect(a).toThrowError();}

\end{DoxyCode}


The following patterns are not considered warnings\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{expect(a).toHaveBeenCalled();}
\DoxyCodeLine{expect(a).toHaveBeenCalledTimes();}
\DoxyCodeLine{expect(a).toHaveBeenCalledWith();}
\DoxyCodeLine{expect(a).toHaveBeenLastCalledWith();}
\DoxyCodeLine{expect(a).toHaveBeenNthCalledWith();}
\DoxyCodeLine{expect(a).toHaveReturned();}
\DoxyCodeLine{expect(a).toHaveReturnedTimes();}
\DoxyCodeLine{expect(a).toHaveReturnedWith();}
\DoxyCodeLine{expect(a).toHaveLastReturnedWith();}
\DoxyCodeLine{expect(a).toHaveNthReturnedWith();}
\DoxyCodeLine{expect(a).toThrow();}

\end{DoxyCode}
