\begin{DoxyItemize}
\item Seamless typescript and ESM syntax support
\item Seamless interoperability between ESM and Common\+JS
\item Synchronous API to replace {\ttfamily require}
\item Super slim and zero dependency
\item Smart syntax detection to avoid extra transforms
\item Common\+JS cache integration
\item Filesystem transpile hard cache
\item V8 compile cache
\item Custom resolve alias
