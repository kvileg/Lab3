 \href{https://github.com/postcss/postcss}{\texttt{ Post\+CSS}} plugin to discard overridden {\ttfamily @keyframes} or {\ttfamily @counter-\/style}.

{\ttfamily @keyframes} or {\ttfamily @counter-\/style} will be overridden by those who share the same identifiers and appear later in stylesheets. So we can discard all of them except the last one. When defined inside a {\ttfamily @media} or {\ttfamily @supports} rule, {\ttfamily @keyframes} and {\ttfamily @counter-\/style} rules only override global rules in some of the client browsers so they need handled separately. This plugin has taken care of this and transforms the Post\+Css AST {\bfseries{safely}}.
