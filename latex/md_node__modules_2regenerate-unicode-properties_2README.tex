 {\itshape regenerate-\/unicode-\/properties} is a collection of \href{https://github.com/mathiasbynens/regenerate}{\texttt{ Regenerate}} sets for \href{https://github.com/tc39/proposal-regexp-unicode-property-escapes}{\texttt{ various Unicode properties}}.\hypertarget{md_node__modules_2regenerate-unicode-properties_2README_autotoc_md23080}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_node__modules_2regenerate-unicode-properties_2README_autotoc_md23080}
To use {\itshape regenerate-\/unicode-\/properties} programmatically, install it as a dependency via \href{https://www.npmjs.com/}{\texttt{ npm}}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ npm\ install\ regenerate-\/unicode-\/properties}

\end{DoxyCode}
\hypertarget{md_node__modules_2regenerate-unicode-properties_2README_autotoc_md23081}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md_node__modules_2regenerate-unicode-properties_2README_autotoc_md23081}
To get a map of supported properties and their values\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ properties\ =\ require('regenerate-\/unicode-\/properties');}

\end{DoxyCode}


To get a specific Regenerate set\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ Examples:}
\DoxyCodeLine{const\ Lu\ =\ require('regenerate-\/unicode-\/properties/General\_Category/Uppercase\_Letter.js').characters;}
\DoxyCodeLine{const\ Greek\ =\ require('regenerate-\/unicode-\/properties/Script\_Extensions/Greek.js').characters;}

\end{DoxyCode}


Some properties can also refer to strings rather than single characters\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{const\ \{\ characters,\ strings\ \}\ =\ require('regenerate-\/unicode-\/properties/Property\_of\_Strings/Basic\_Emoji.js');}

\end{DoxyCode}


To get the Unicode version the data was based on\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ unicodeVersion\ =\ require('regenerate-\/unicode-\/properties/unicode-\/version.js');}

\end{DoxyCode}
