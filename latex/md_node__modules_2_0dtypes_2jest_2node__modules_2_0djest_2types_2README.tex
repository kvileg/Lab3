 This package contains shared types of Jest\textquotesingle{}s packages.

If you are looking for types of \href{https://jestjs.io/docs/api}{\texttt{ Jest globals}}, you can import them from {\ttfamily @jest/globals} package\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{import\ \{describe,\ expect,\ it\}\ from\ '@jest/globals';}
\DoxyCodeLine{}
\DoxyCodeLine{describe('my\ tests',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ it('works',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ \ \ expect(1).toBe(1);}
\DoxyCodeLine{\ \ \});}
\DoxyCodeLine{\});}

\end{DoxyCode}


If you prefer to omit imports, a similar result can be achieved installing the \href{https://npmjs.com/package/@types/jest}{\texttt{ @types/jest}} package. Note that this is a third party library maintained at \href{https://github.com/DefinitelyTyped/DefinitelyTyped/tree/master/types/jest}{\texttt{ Definitely\+Typed}} and may not cover the latest Jest features.

Another use-\/case for {\ttfamily @types/jest} is a typed Jest config as those types are not provided by Jest out of the box\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ jest.config.ts}
\DoxyCodeLine{import\ \{Config\}\ from\ '@jest/types';}
\DoxyCodeLine{}
\DoxyCodeLine{const\ config:\ Config.InitialOptions\ =\ \{}
\DoxyCodeLine{\ \ //\ some\ typed\ config}
\DoxyCodeLine{\};}
\DoxyCodeLine{}
\DoxyCodeLine{export\ default\ config;}

\end{DoxyCode}
